Test Overview

This test demonstrates the new soft transition functionality in NDF Studio. Transitions are now simulated in-memory only, without affecting the database.

Test Steps

  1. Navigate to the graph: Go to the NDF Studio frontend and load a graph with transitions
  2. Identify transition nodes: Look for diamond-shaped nodes (transitions) - they should show a pointer cursor on hover
  3. Click a transition: Click on the "oxygen ionization" transition node
  4. Observe changes:
    • Loading message appears: "Simulating transition..."
    • Success message: "Transition simulated! (In-memory only)"
    • Yellow simulation mode indicator appears
